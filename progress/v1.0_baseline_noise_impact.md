# v1.0 — Baseline: Impact of Observation Noise on Robot Manipulation

**Date:** 2026-02-27
**Goal:** Establish baselines showing how observation noise degrades a pretrained policy.

## Setup
- **Task:** `Isaac-Lift-Cube-Franka-v0` (Franka Panda, lift cube to target height)
- **Policy:** Pretrained RSL-RL PPO checkpoint (published by Isaac Lab)
- **Actor:** MLP 36 → 256 → 128 → 64 → 8, ELU activations
- **Observation (36-dim):** joint_pos[0:9], joint_vel[9:18], object_pos[18:21], target_pose[21:28], prev_actions[28:36]
- **Environments:** 32 parallel
- **Episodes per test:** 40-80

## Noise Levels Tested
| Level    | joint_pos_std | joint_vel_std | object_pos_std |
|----------|--------------|--------------|----------------|
| None     | 0.0          | 0.0          | 0.0            |
| Low      | 0.005        | 0.02         | 0.02           |
| Medium   | 0.01         | 0.05         | 0.05           |
| High     | 0.02         | 0.1          | 0.10           |
| Extreme  | 0.05         | 0.2          | 0.15           |

## Results — Success Rate vs Noise

| Noise Level | Success Rate | Avg Reward |
|-------------|-------------|------------|
| None        | 100%        | ~155       |
| Low         | ~98%        | ~145       |
| Medium      | 92.5%       | 116.7      |
| High        | 58.0%       | 45.8       |
| Extreme     | ~17%        | ~7.0       |

**Key finding:** High noise (10 cm object position std) drops success from 100% to 58%.
